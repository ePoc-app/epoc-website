## Introduction

Les événements d'apprentissage décrits ci-dessous sont basés sur le standard **[xAPI](https://registry.tincanapi.com/#home/verbs)**. 
Ces événements permettent de suivre les interactions des utilisateurs avec les contenus pédagogiques. Ils peuvent 
être utilisés par le **[système de badges](./evaluation#badge)** pour récompenser les progrès, par le **système de verrouillage des contenus**
pour contrôler l'accès aux ressources, et prochainement pour les **scénarios à embranchements** afin de définir des 
conditions d'obtention ou de déblocage.


## Définition des événements d'apprentissage

### Chapitres
::stack{class="p-4"}
    ::field-group
        ::field{name="started" type="boolean"}
        Vrai lorsqu'un utilisateur lance un chapitre.
        ::
    ::
::

### Pages
::stack{class="p-4"}
    ::field-group
        ::field{name="viewed" type="boolean"}
        Vrai lorsqu'un utilisateur consulte une page qu'il n'a pas encore vue.
        ::
    ::
::

### Texte
::stack{class="p-4"}
    ::field-group
        ::field{name="read" type="boolean"}
        Vrai lorsqu'un utilisateur consulte une page de texte.
        ::
    ::
::

### Audio
::stack{class="p-4"}
    ::field-group
        ::field{name="played" type="boolean"}
        Vrai lorsqu'un utilisateur lance un podcast audio.
        ::
        
        ::field{name="listened" type="boolean"}
        Vrai lorsqu'un utilisateur a écouté au moins 50% d'un podcast audio.
        ::
    ::
::

### Vidéo
::stack{class="p-4"}
    ::field-group
        ::field{name="played" type="boolean"}
        Vrai lorsqu'un utilisateur lance une vidéo.
        ::
        
        ::field{name="watched" type="boolean"}
        Vrai lorsqu'un utilisateur a regardé au moins 50% d'une vidéo.
        ::
    ::
::

### Évaluation
::stack{class="p-4"}
    ::field-group
        ::field{name="started" type="boolean"}
        Vrai lorsqu'un utilisateur lance une activité d'évaluation.
        ::
        
        ::field{name="completed" type="boolean"}
        Vrai lorsqu'un utilisateur termine une activité d'évaluation.
        ::
        
        ::field{name="scored" type="number"}
        Score de l'utilisateur qui termine une activité d'évaluation.
        ::
    ::
::

### Questions
::stack{class="p-4"}
    ::field-group
        ::field{name="attempted" type="boolean"}
        Vrai lorsqu'un utilisateur répond à une question.
        ::
        
        ::field{name="scored" type="number"}
        Score de l'utilisateur qui répond à une question.
        ::
        
        ::field{name="passed" type="boolean"}
        Vrai lorsqu'un utilisateur répond juste à une question.
        ::
    ::
::
